# Интерпретатор Brainfuck

Brainfuck – один из эзотерических языков программирования. Был придуман в 1993 году Урбаном Мюллером. 
Язык имеет всего восемь команд, каждая из которых записывается одним символом. 
Исходный код программы на Brainfuck представляет собой последовательность этих символов без какого-либо дополнительного синтаксиса.
Несмотря на внешнюю примитивность, Brainfuck с бесконечным набором ячеек имеет тьюринговскую полноту, а, следовательно, по потенциальным возможностям не уступает языкам, подобным C, Pascal, Java и др.

- Вычисления осуществляются на ленте памяти содержащей `30 000` ячеек.
- В начальном состоянии указатель находится в крайней левой позиции, а все ячейки заполнены нулями. 
- Увеличение или уменьшение значений ячеек происходит по модулю 256. 
- Каждая ячейка должна содержать беззнаковый байт с поведением переноса (т. е. 255 + 1 = 0, 0 - 1 = 255), инициализированный значением 0.

Нужно написать интерпретатор этого языка, который будет поддерживать следующие инструкции:

- `>` – увеличить указатель (чтобы он указывал на следующую ячейку справа),
- `<` – уменьшить указатель (чтобы он указывал на следующую ячейку слева),
- `+` – увеличить значение в текущей ячейке на единицу,
- `-` – уменьшить значение в текущей ячейке на единицу,
- `.` – вывести байт по указателю (напечатать значение из текущей ячейки),
- `,` – принять один байт из входных данных и сохранить его значение в текущей ячейке,
- `[` – если байт по указателю (то есть в текущей ячейке) равен нулю, то перейти вперед по тексту программы на символ, следующий за соответствующей `]` (с учетом вложенности скобок),
- `]` – если байт по указателе (то есть в текущей ячейке) ненулевой, то перейти назад по тесту программы на символ `[` (с учетом вложенности скобок).

Предполагаем, что в коде исходной программы на Brainfuck ошибок нет, но если вдруг обнаружится ошибка в ходе выполнения программы (например, закончится лента памяти или обнаружатся непарные квадратные скобки), то нужно вывести на экран сообщение 'ERROR' и остановить интерпретацию программы.

В командной строке при вызове программы указываются имена двух файлов: файла с программой на языке Brainfuck и файла со входными данными.
